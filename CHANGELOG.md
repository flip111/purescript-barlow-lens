# Changelog

## v0.5.0
- Simplified parsing 
- Removed requirement for records. A lens can now start e.g. with `?` 

## v0.4.0
- Support for `Newtype` using `!`

## v0.3.0
- Support for `Either` using `<` and `>`
- Support for `Array` and other `Traversal` using `+`

## v0.2.0
- Support for `Maybe` using `?`

## v0.1.0

- Support for records
